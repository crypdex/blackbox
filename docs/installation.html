Installation

Prerequisites

Getting Blackbox setup is fairly straightforward, but does require a few software prerequisites.

  • Docker
  • Docker Compose

If you are developing on macOS, Docker Desktop for Mac has everything you need including Docker Compose. For Linux on ARM64, Docker Compose should be installed via pip because Compose does not have an ARM64 build.

Install Docker the EZ way

$ curl -fsSL https://get.docker.com -o get-docker.sh
$ sh get-docker.sh

Docker Compose ARM64 (Armbian)

Installing Docker-Compose on ARM devices should be done via python/pip

$ apt install libffi-dev python3-dev python3-pip
$ pip3 install wheel setuptools
$ pip3 install docker-compose

Install BlackboxOS

Blackbox is distributed via APT for Linux and Homebrew on macOS.

Linux

Add the Blackbox APT repo to your sources

# Do this once
$ echo "deb [trusted=yes] https://apt.fury.io/crypdex/ /" > /etc/apt/sources.list.d/fury.list

Install

# Install Blackbox
$ apt update && apt install blackbox-os

macOS

$ brew install crypdex/blackbox/blackbox-os

Manual Package Installation

Download the most current package from the releases and use apt to install. If it is already installed, it will be updated it.
